Mahlzeit kopieren

Möchtes du von dieser Mahlzeit eine Kopie erstellen?

Rezepte dieser Mahlzeit werden dabei nicht kopiert, sondern nur verknüpft. Alternativ kannst du von jedem Rezept eine zusätzliche Kopie erstellen lassen (nicht empfohlen)! Mache dies nur, wenn du keinen Schreib-Zugriff auf die alle verwendeten Rezepte hast.

Möchtes du von dieser Mahlzeit eine Kopie erstellen?

Diese Mahlzeit ist eine Vorlage. Du kannst sie bearbeiten, indem du eine Kopie erstellst. Rezepte, die ebenfalls aus Vorlagen erstellt worden sind, werden dabei automaisch verknüpft.

Alternativ kannst du von jedem Rezept eine zusätzliche Kopie erstellen lassen (nicht empfohlen).
