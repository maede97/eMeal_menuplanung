Zutaten aus Excel importieren (Experimentelles Feature)

Über dieses Fenster können Zuateten direkt aus einem Excel-File eingefügt werden.

Damit der Import fehlerfei funktioniert, es es wichtig, dass dein Excel folgende drei Spalten hat: Menge pro Person, Einheit, Zutat. Nun kannst du die Zellen anwählen, kopieren und im folgenden Feld einfügen.
